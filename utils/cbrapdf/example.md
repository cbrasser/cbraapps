# Markdown to PDF with Glow Styling

This is a **sample markdown document** to demonstrate the conversion to PDF using Glow's beautiful styling.

## Features

The `md2pdf` script supports:

- **Dark theme** (default)
- **Light theme**
- Automatic output filename generation
- Beautiful syntax highlighting

## Code Example

Here's some Python code:

```python
def hello_world():
    """Print a greeting message."""
    message = "Hello, World!"
    print(message)
    return True

if __name__ == "__main__":
    hello_world()
```
